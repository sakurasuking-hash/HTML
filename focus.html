模块说明

本页面对应 PRD 中的「四、后台界面交互」，用于向研发展示：

  • 入口位置与权限要求
  • 列表页字段、筛选、排序与操作
  • 新建 / 编辑页的字段布局
  • 发布 / 下线 / 删除的确认弹窗
  • 预览弹窗：仅作为容器，不包含示例文案

本页不接真实接口，仅为结构和样式示意。

4.1 入口（优化现有产品结构）

菜单入口位置

在投顾后台「内容管理」模块下新增子菜单「重点关注品种」。

后续将「期市聚焦」等产品也列到二级菜单展示，各产品内容分开管理。

访问权限

功能入口仅对具备「策略内容管理」权限的运营人员开放。

4.4 操作弹窗与交互 交互说明

  • 发布 / 下线确认弹窗：在单条或批量发布 / 下线前，给出操作影响提示。
  • 删除确认弹窗：仅草稿 + 已下线可删除，操作前二次确认，删除后不可恢复。
  • 预览弹窗：用于预览 C 端展示样式，本示例中仅保留空容器，不展示具体内容文案。
